## Border

The `border` utility is used to add border to an element. It can be used to add border to all sides or to a specific side of an element.

<div class="border-light br-8 p1">
  This div has a light border around all sides.
</div>

<div class="border-bottom-light br-8 p1">
  This div has a light border only at the bottom.
</div>

<div class="border-top-light br-8 p1">
  This div has a light border only at the top.
</div>

<div class="border-left-light br-8 p1">
  This div has a light border only on the left side.
</div>

<div class="border-right-light br-8 p1">
  This div has a light border only on the right side.
</div>

<div class="border-primary br-8 p1">
  This div has a primary color border around all sides.
</div>

<div class="border-spot br-8 p1">
  This div has a spot color border around all sides.
</div>

<div class="border-success br-8 p1">
