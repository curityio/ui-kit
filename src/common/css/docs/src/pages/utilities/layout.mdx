# Layout Utilities

Layout utilities provide classes for controlling common layout properties including display, position, overflow, and z-index.

## Display

Control the display property of elements.

- `.d-none` - Sets display to none
- `.d-block` - Sets display to block
- `.d-inline` - Sets display to inline
- `.d-inline-block` - Sets display to inline-block
- `.d-flex` - Sets display to flex
- `.d-inline-flex` - Sets display to inline-flex
- `.d-grid` - Sets display to grid

## Position

Control the position property of elements.

- `.position-static` - Sets position to static
- `.position-relative` - Sets position to relative
- `.position-absolute` - Sets position to absolute
- `.position-fixed` - Sets position to fixed
- `.position-sticky` - Sets position to sticky

## Overflow

Control how content overflow is handled.

- `.overflow-auto` - Sets overflow to auto
- `.overflow-hidden` - Sets overflow to hidden
- `.overflow-visible` - Sets overflow to visible
- `.overflow-scroll` - Sets overflow to scroll

## Z-Index

Control the stacking order of positioned elements.

- `.z-0` - Sets z-index to 0
- `.z-10` - Sets z-index to 10
- `.z-20` - Sets z-index to 20
- `.z-30` - Sets z-index to 30
- `.z-40` - Sets z-index to 40
- `.z-50` - Sets z-index to 50

## Visibility

Control element visibility.

- `.visible` - Sets visibility to visible
- `.invisible` - Sets visibility to hidden (element still takes up space)

## Float

Control element float behavior.

- `.float-start` - Float element to the start (left in LTR)
- `.float-end` - Float element to the end (right in LTR)
- `.float-none` - Remove float

## Clear

Clear floated elements.

- `.clearfix` - Clear floats using pseudo-element
- `.clear-both` - Clear both left and right floats
