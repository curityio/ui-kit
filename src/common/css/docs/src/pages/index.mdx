## Use in Curity UI Kit repository

```shell
import "@curity-ui-kit/css/lib/dist/index.css";
```

## Use via GitHub Packages

To use this package outside externally using GitHub Packages.


Install

First add `@curityio:registry=https://npm.pkg.github.com` to `.npmrc` file in your project root.

Then install the package

```shell
npm install @curityio/css
```

Use

```shell
import "@curityio/css/lib/dist/index.css";
```
