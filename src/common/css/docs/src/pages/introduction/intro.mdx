# Intro

Curity CSS is a simple and easy to use CSS library for building user interfaces. It is built with modern CSS features like Flexbox and CSS Grid, and is designed to be mobile friendly by default.

The library provides a set of CSS classes for common UI components like buttons, forms, cards, and navigation, as well as layout utilities for creating responsive layouts.

## Benefits

- **Theming**: Easy customization through CSS variables defined in `src/common/css/lib/src/curity-theme.css`
- **Mobile-First**: Responsive design built-in with mobile-friendly defaults
- **Modern CSS**: Leverages Flexbox and CSS Grid for flexible layouts
- **Rapid Prototyping**: Pre-built UI components for faster development
- **Lightweight**: Simple, minimal footprint with no dependencies
