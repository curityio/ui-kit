# Theming

[CSS custom properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties) is used for theming.

This means you can create a theme only by changing variable values. This is the same structure we use for Curity UI Builder and the when creating themes inside Curity Admin UI.

In `src/curity-theme.css` you can find all the variables that can be changed. To set a variable, you can scope it to `:root` which will make it avaliable everywhere, suited for global theming.

```css
:root {
  --type-base-size: 1rem;
}
```
You can also scope to a specific selector. This is powerful since you can use the native cascade and reuse variables.

```css
main {
  --type-base-size: 1rem;
}

```
