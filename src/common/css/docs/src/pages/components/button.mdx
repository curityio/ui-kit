# Buttons

Both regular `<button>` and `<a>` tags can be styled using the button classes.

## Filled

<div>
  <button class="button button-tiny button-primary">Button primary tiny</button>
</div>
<div>
  <button class="button button-small button-primary">Button primary small</button>
</div>
<div>
  <button class="button button-medium button-primary">Button primary medium</button>
</div>
<div>
  <button class="button button-large button-primary">Button primary large</button>
</div>

```html
<button class="button button-tiny button-primary">Button primary tiny</button>
<button class="button button-small button-primary">Button primary small</button>
<button class="button button-medium button-primary">Button primary medium</button>
<button class="button button-large button-primary">Button primary large</button>
```
