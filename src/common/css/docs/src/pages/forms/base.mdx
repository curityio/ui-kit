# Base

## Form example

The `.form-field` class can wrap form elements to provide a consistent look and feel. The `.label` class is used to style the label element, and the `.field` class is used to style the input element. The `.w100` class is used to make the input element take up the full width of its parent container.

<form className="mx-auto mw-40 border-light p4 br-8 mt4 mb4">
  <h2>Sign In</h2>

<div className="form-field">
  <label for="username">Username</label>
  <input className="field w100" type="text" name="username" required autoFocus />
</div>
<div className="form-field">
  <label for="password">Password </label>
  <input className="field w100" type="password" name="password" required />
</div>

  <div className="form-field py2">
  <label for="pet-select">Choose a pet:</label>

    <select name="pets" id="pet-select">
    <option value="">Select pet</option>
    <option value="dog">Dog</option>
    <option value="cat">Cat</option>
    <option value="hamster">Hamster</option>
    <option value="parrot">Parrot</option>
    <option value="spider">Spider</option>
    <option value="goldfish">Goldfish</option>
    </select>

    </div>

<div classname="form-field py2">
  <input className="button button-primary button-medium w100" type="submit" value="Login" />
</div>
