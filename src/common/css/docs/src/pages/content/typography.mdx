## Lists

### Unordered List

- List item 1
- List item 2
- List item 3
  - Nested item 1
  - Nested item 2

```html
<ul>
  <li>List item 1</li>
  <li>List item 2</li>
  <li>List item 3
    <ul>
      <li>Nested item 1</li>
      <li>Nested item 2</li>
    </ul>
  </li>
</ul>
```

### Ordered List

1. First item
2. Second item
3. Third item
   1. Nested first
   2. Nested second

```html
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item
    <ol>
      <li>Nested first</li>
      <li>Nested second</li>
    </ol>
  </li>
</ol>
```
