# Grid (legacy)

<div class="alert alert-warning"><p>This is an older grid system using floats. It is still in the CSS Toolkit but is not recommended to use anymore. Use flexbox or CSS Grid instead.</p></div>



### Usage

```html
<div class="col col-6">Column 6/12 of the width</div>
```
