# Whitespace

Utilities for controlling an element's white-space property.

The `--space-1` to `--space-4` is used for spacing, to keep a vertical rythm. The space variables is using rems units and starts from 0.5 rems.

## Classes

#### Margin

+ `m0` - No margin
+ `mt0` - No top margin
+ `mt1` - 1 unit margin above
+ `mt2` - 2 units margin below
+ `mt3` - 3 units margin below
+ `mt4` - 4 units margin below
+ `ml1` - 1 unit margin left
+ `ml2` - 2 unit margin left
+ `ml3` - 3 unit margin left
+ `ml4` - 4 unit margin left
+ `mr1` - 1 unit margin right
+ `mr2` - 2 unit margin right
+ `mr3` - 3 unit margin right
+ `mr4` - 4 unit margin righ
+ `mx-auto` - center a block leven element using `margin: 0 auto`

#### Padding

+ `p0` - No padding
+ `pt0` - No top padding
+ `pt1` - 1 unit padding above
+ `pt2` - 2 units padding below
+ `pt3` - 3 units padding below
+ `pt4` - 4 units padding below
+ `py1` - 1 unit padding top and bottom
+ `py2` - 2 unit padding top and bottom
+ `py3` - 3 unit padding top and bottom
+ `py4` - 4 unit padding top and bottom
+ `m-unset` - Use `margin: unset`
+ `p-unset` - Use `padding: unset`
