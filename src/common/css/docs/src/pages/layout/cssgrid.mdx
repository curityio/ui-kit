# CSS Grid

CSS Grid is the newest spec in CSS for  two-dimensional grid-based layout systems. Flexbox is also great but its one-directional flow has different use cases.

Curity CSS comes with helper classes for CSS Grid

```html
<div class="grid-container p4" columns="1" sm-columns="2" md-columns="3" gap="3">
  <div>Child</div>
  <div>Child</div>
  <div>Child</div>
</div>
```

<div class="grid-container p4 demo-element" columns="1" sm-columns="2" md-columns="3" gap="3">
  <div class="demo-element">Child</div>
  <div class="demo-element">Child</div>
  <div class="demo-element">Child</div>
</div>
